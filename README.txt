# DungeonRPG

３Dでのダンジョン探索と、3Dのアクション要素を組み合わせたゲームを作成しました。


起動方法

「ManagedDxlGame.exe - ショートカット」から起動できます。


・ダンジョン探索シーン：
	WASDで移動、回転しながらダンジョンを探索するシーンです。
	移動している敵キャラクターと接触するとバトルシーンに移行します。
	マップ内に配置されているゴールまで行くとゲームクリアです。

・バトルシーン：
	敵キャラクターとバトルを行うシーンです。
	WASDで移動、マウス左クリックで向いている向きに攻撃します。
	敵キャラクターまたは自分の体力を削りきると終了します。


工夫した点
・ダンジョンシーンにおいて、一気に移動を行うのではなく、
　徐々にマスを移動するようにした点
・ダンジョンにおいて、敵キャラクターをどの向きから見てもプレイヤーの向きを向くようにした点

・また、プログラムに関する内容ではありませんが、ゲーム内の素材の多くを自身で作成しました。
